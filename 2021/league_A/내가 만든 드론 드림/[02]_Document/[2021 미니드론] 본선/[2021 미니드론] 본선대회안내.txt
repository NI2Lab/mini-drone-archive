# 본선대회안내
1) 대회당일 실행파일
- 각 팀 Github Repository Clone 후 main.m 파일 실행예정
- 해당 파일 없을 시, 기권으로 간주
- 코드 실행 안될 시, 미션 실패로 간주

2) 대회진행순서
- A리그 B리그 번갈아가며 한 팀씩 진행
- 각 리그 팀 순서는 무작위 랜덤
- 1차시도, 2차시도 총 2번의 기회

3) 맵 규격
- 천막에 링이 있는 구조
- 링은 상하좌우로 이동될 수 있음
- 링의 개수는 총 3개
- 링의 위치는 1차, 2차에서 서로 다르게 위치함
- 표식은 링 뒤에 위치, 표식에 따라 드론제어 결정
- ex) 빨간색: 90도 좌회전, 파란색: 착륙 등
- 상세규격은 추후공지

4) 본선평가방식
- 각 단계별 Cut-Off Time 측정, 시간초과 시 테스트종료
- 각 단계별 시간 측정(1, 2단계는 링 통과시간, 3단계는 착지시간)
- 제한시간초과시 통과한 단계에 대해서만 기록 및 평가
- 1순위 평가기준: 몇 단계까지 통과했는지
- 2순위 평가기준: 통과단계 동일 시 1, 2차 통과횟수평가
- 3순위 평가기준: 통과단계 및 통과횟수 동일 시, 시간평가
- 4순위 평가기준: 상기 평가기준 모두 동일 시, 1차시도 통과에 가점
