# Github 기초사용법
1. 초기환경세팅(최초 1회만 수행)
ㄴ 1) 새 폴더 생성
ㄴ 2) 폴더 주소창에 cmd 입력
ㄴ 3) git clone https://github.com/JoSihun/DroneDream 입력
2. 실질적인 사용법
ㄴ 1) git pull
ㄴ 2) 파일 수정
ㄴ 3) git add .
ㄴ 4) git commit -m "업데이트사항입력"
ㄴ 5) git push
3. 파일 수정/삭제 주의사항
ㄴ 1) 내가 쉬는 동안 다른 사람이 파일을 업데이트했을 수도 있으므로 반드시 git pull을 입력후 파일을 수정하세요.
ㄴ 2) 공동개발환경이므로 한 사람이 파일을 수정/삭제하고 git push 하면, 나머지 사람도 파일이 수정/삭제됨에 주의하세요.
